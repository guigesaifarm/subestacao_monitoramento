Sistema SCADA Web completo para supervisão de subestação elétrica com as seguintes características:
🎯 Funcionalidades Principais
1. Diagrama Unifilar Interativo (Single Line Diagram)
Visualização em tempo real do estado dos equipamentos
Linhas energizadas com animação de fluxo de energia
Disjuntores coloridos (verde=fechado, vermelho=aberto)
Chaves seccionadoras com indicação de posição
Transformadores com monitoramento térmico
2. Painel de Controle do Operador
Comandos de abertura/fechamento de disjuntores
Parada de emergência com confirmação
Status de comunicação de cada IED
Contador de surtos atmosféricos (SPDA)
3. Monitoramento em Tempo Real
Medições TC/TP (corrente e tensão trifásicas)
Gráfico de tendências atualizado a cada segundo
Temperatura e carga dos transformadores
Frequência e fator de potência
4. Sistema de Proteção (IED)
Indicação das funções de proteção ativas (50/51, 87T, etc.)
Status de comunicação IEC 61850
Alarmes de falha e anomalias
5. Registro de Eventos
Log cronológico de todas as operações
Categorização por cores (verde=normal, amarelo=alerta, vermelho=critical)
Timestamps precisos
🎨 Design Técnico
Interface escura (padrão industrial) para operação prolongada
Glassmorphism e transparências para visual moderno
